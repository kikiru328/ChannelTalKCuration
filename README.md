# Flask

	Webhook API 를 전송하는 타 웹 (channel talk) 과 Flask를 연동하여
        24시간 GET/POST를 진행함에 있어 불필요하던 작동을 WEBHOOK API로 간단하게 구현하게 함.
	하지만 24/7 작동을 위해 AWS LIGHTSAIL에 Flask를 Nohup으로 실행하여 24/7 작동구현됨.

> Tutorial은 Ngrok 서비스를 이용하였다.
